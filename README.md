## üìä Performance Metrics to Track

### Essential KPIs for Your Deriv Bot:

**1. Win Rate**
- Target: 55%+ for most strategies
- Higher is better, but focus on profitability

**2. Profit Factor**
- Gross profit √∑ Gross loss
- Target: 1.5+ (means you make $1.50 for every $1 lost)

**3. Maximum Drawdown**
- Largest peak-to-trough decline
- Target: <30% of account balance

**4. Average Win/Loss Ratio**
- Average winning trade √∑ Average losing trade
- Target: 1.5+ (winners should be bigger than losers)

**5. Sharpe Ratio**
- Risk-adjusted returns
- Target: 1.0+ (higher is better)

**6. Expectancy**
- Expected profit per trade
- Formula: (Win% √ó Avg Win) - (Loss% √ó Avg Loss)
- Target: Positive value

**7. Consecutive Loss Streak**
- Maximum losses in a row
- Monitor carefully‚Äîadjust risk if this increases
